## 目录
| **名称** | **作用** | 
| :-----| :----- |
| **core** | 使用Rust构建的 arbitrage 框架核心. |  
| **jdk** | 基于arbitrage core构建的Java SDK. |  
| **py** | 基于arbitrage core构建的Python SDK. |  
| **terminal** | 基于arbitrage core构建的金融控制台终端程序. |  
| **desktop** | 基于arbitrage core构建的桌面应用程序. |  
| **website** | arbitrage core官方网站. |
